2. `async def generate_backchannel(user_text: str, category: str = "", last_assistant_text: str = "") -> str | None`:
   - Calls Haiku (`claude-haiku-4-5-20251001`) via Messages API
   - System prompt (keep short to minimize latency):
     ```
     You generate ultra-short backchannel responses (1-5 words) to acknowledge what someone just said before a fuller response arrives. Match the tone: casual for chat, attentive for questions, energetic for tasks. Never use filler words like "um" or "uh". Just the words, no quotes or punctuation except ! or ?.      ```
     ```
   - User message: `user_text` (just the transcript, keep it minimal)
   - If `last_assistant_text` is provided, include a one-line assistant context: prepend `[Context: you just said "${last_assistant_text[-100:]}"]\n` to the user message

   - Return None on TTS failure
   - Import PIPER_CMD and PIPER_MODEL paths using same pattern as clip_factory.py (Path.home() based)
